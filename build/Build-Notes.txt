To compile in windows:
1. Install Node
2. Clone the homebridge-vivint repository
3. Navigate to the homebridge-vivin\build directory in the command shell
4. Run: package-homebridge-vivint.bat

To publish to NPM
1. Navigate to homebridge-vivint\release-homebridge-vivint
2a.If not logged into NPM, Run: npm login
2b. Run: npm publish
